# 🌍 SDG 4: Quality Education

This is a responsive and informative webpage designed to promote **Sustainable Development Goal 4 (SDG 4)** — _"Ensure inclusive and equitable quality education and promote lifelong learning opportunities for all."_  

The page highlights key initiatives that support quality education using clean HTML and CSS, along with accessible visuals and interactive navigation.

---

## 🚀 Features

- 🌐 Clean and responsive layout
- 📱 Can be viewed on all screen sizes (mobile, tablet, desktop)
- 🎯 Focused content on SDG 4
- 🧭 Smooth navigation between sections
- 🎨 Custom styling with CSS3
- ⚙️ Basic interactivity using JavaScript

---

## 📂 Project Structure
```
sdg4-quality-education/
├── index.html # Main HTML file
├── styles.css # External CSS file for styling
├── script.js # javascript file
└── README.md # Project documentation
```

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Transitions)
- Javascript 
- Google Fonts (Montserrat)

## 🚀 Getting Started

Follow these simple steps to clone and run this project locally:

### 📥 Clone the Repository

```bash
git clone https://github.com/your-username/sdg4-quality-education.git
```

### 📂 Navigate into the Project Folder
```bash
cd sdg4-quality-education
```
🌐 Open in Browser
You can open index.html directly in your web browser:

On Windows: Double-click the index.html file

Or use VS Code Live Server:

Open folder in VS Code

Right-click index.html → “Open with Live Server”
